# rabbitmq

1. pull image

   ```shell
   docker pull rabbitmq:3.8.17-management
   ```

   

2. git clone

   ```shell
   git clone https://github.com/fangxh2013/rabbitmq-docker-compose.git
   ```

   

3. 进入目录，运行：

```shell
docker-compose up -d
```



4. 浏览器中输入以下url进入管理界面：

   ```shell
   http://localhost:15672/
   ```
